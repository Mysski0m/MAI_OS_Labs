\section{Результаты}

Получены 5 файлов, содержащих /texttt{strace}-логи по лабораторным работам:
\begin{itemize}
    \item по одному файлу для лабораторных работ №1, №2 и №3;
    \item для лабораторной работы №4 приложены 2 файла: один для тестовой программы, использующей линковку времени компиляции (\texttt{main\_link}), и один для программы, выполняющей загрузку библиотек во время выполнения (\texttt{main\_runtime}).
\end{itemize}
Все логи получены с использованием флага \texttt{-f} (где применимо), что позволило зафиксировать системные вызовы как основного, так и дочерних потоков/процессов. Анализ подтвердил корректное использование требуемых механизмов ОС: межпроцессного взаимодействия через каналы и сигналы (ЛР 1, 3), многопоточности (ЛР 2) и динамической загрузки библиотек (ЛР 4).
